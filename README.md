# Compialdor Pascal
Um compilador (em desenvolvimento) para Pascal, feito em Ruby, para a matéria de Compialdores do curso de Ciência da Computação da UNESP - Bauru.


## Uso
Em um terminal, executar

```
ruby main.rb <nome_do_arquivo.pas>
```
